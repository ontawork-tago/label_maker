# ID Label Production Management Guide

생산 배치마다 값이 달라지는 항목입니다. 최종 엑셀에서 각 향지별 `_Val` 컬럼을 수정해야 합니다.

| b       | 기기의명칭 (규격모델명)    | 생산 시 매번 기입 필요 (S/N, 날짜 등) |
| c       | 모델명칭             | 생산 시 매번 기입 필요 (S/N, 날짜 등) |
| e       | 일련번호             | 생산 시 매번 기입 필요 (S/N, 날짜 등) |
